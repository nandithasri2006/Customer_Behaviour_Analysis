# Customer_Behaviour_Analysis
Customer Shopping Behavior Analysis
Overview

This project analyzes customer shopping behavior using transactional data (3,900 records) to uncover patterns in spending, product preferences, and purchase trends.

The goal is to generate data-driven insights that help improve marketing strategies, customer engagement, and overall business performance.

 Objective

Analyze customer purchase behavior

Identify key drivers of sales (discounts, reviews, etc.)

Understand customer segmentation and loyalty

Compare different shopping patterns

 Dataset

Records: 3,900

Features: 18

Includes:

Customer demographics (Age, Gender, Location, Subscription)

Purchase details (Item, Category, Amount, Season)

Behavioral data (Discount, Promo Code, Frequency, Ratings)

Missing Values: 37 in Review Rating

 Tech Stack

Python → Data Cleaning & EDA

SQL → Data Analysis

Power BI → Dashboard & Visualization

 Key Analysis

Customer segmentation

Sales trend analysis

Product category performance

Discount & promo impact

Repeat vs new customers

 Key Insights

Discounts significantly influence purchasing decisions

A small group of customers contributes most of the revenue

Repeat customers are key to business growth

Seasonal trends impact sales patterns

 Recommendations

Target high-value customers with personalized offers

Optimize discount strategies

Focus on top-performing product categories

Improve customer retention programs
